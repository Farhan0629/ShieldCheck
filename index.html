About This Project

I Farhan Naser built this project to showcase my understanding of password security, k-Anonymity, SHA-1 hashing, web cryptography, and cybersecurity concepts.

All analysis happens locally in your browser — your password never leaves your device. This demonstrates zero-knowledge security principles.
